# ETL (Extract Transform Load)

Just for fun...

## Install

```
git clone https://github.com/jlg-formation/tsr-nov-2021
cd tsr-nov-2021
npm install
```

## Usage

### development

```
cd tsr-nov-2021
npm start
```

### production

```
cd tsr-nov-2021
npm run build
npm run start:prod
```
